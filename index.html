El proyecto final seria una aplicacion para realizar presupuesto de un trabajo, dividiendo los materiales y la mano de obra.

Ingreso de los materiales que se van a utilizar para armar el presupuesto

Se ingresa descripcion

Se ingresa el tipo de material (se deben cargar los validos que se indica en la ventana del prompt ya que se realiza validacion del mismo)

Se ingresa el proveedor (se deben cargar los validos que se indica en la ventana del prompt ya que se realiza validacion del mismo)

Se ingresa las unidades

Se ingresa el precio unitario

Se ingresa el % de Iva (se deben cargar los validos que se indica en la ventana del prompt ya que se realiza validacion del mismo)

Una vez ingresados se realiza la validacion de todos los ingresos y se indica si alguno esta mal, teniendo que volver a ingresar desde el comienzo

Para Seguir ingresando se pone S , sino sale del ingreso de materiales

Ingresa las tareas a realizar para armar el presupuestoo

Se ingresa descripcion de la tarea

Se ingresa el tipo de la tarea (se deben cargar los validos que se indica en la ventana del prompt ya que se realiza validacion del mismo)

Se ingresan las horas hombre

Se ingresa el valor de la hora hombre

Para Seguir ingresando se pone S , sino sale del ingreso de tareas

Se ingresa el porcentaje de rentabilidad que se quiere tener (uno para materiales y uno para mano de obra)

Luego se sealizan los calculos el valor total del presupuesto

Se puede realizar un filtro de materiales por tipo o por proveedor

Se realizan validaciones de los valores ingresados

La salida de lo ingresado se envia a la consola
